无处不眠

Created: 2023-08-08

《无处不眠》以“在任何地方睡觉”为主题，聚焦城市公共空间中的睡眠现象。我在获得许可后拍摄并录制睡眠者的影像与环境声（车鸣、交通噪音、宠物叫声等），并将不同声音对应到特定情绪与人格。面向城市通勤人群，项目通过屏幕影像与程序交互，把“醒来”的角色场景嵌入城市环境；观众可在屏幕上与声音互动，模拟他们的生存处境。

  • 工具: Processing

沉默的螺旋

Created: 2023-07-18

这个项目以“沉默的螺旋”为切入点，探讨交流中的失衡：一方不断发声，另一方选择沉默。研究发现，这一现象在互联网语境下仍然存在，只是以新的形式延续。作品通过可穿戴装置与舞蹈表演，呈现沉默螺旋背后的社会压力与个体在其中经历的情绪变化，并强调个人主动行动对改变现状的可能性。

  • 工具: Processing, Arduino, 3d打印
